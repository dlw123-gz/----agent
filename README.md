###酒馆战棋多模态决策智能体（Battlegrounds Decision Agent）产品介绍
一、项目概述
本工具是一套基于 VLM 视觉大模型 + LLM 语言大模型构建的炉石传说酒馆战棋全自动决策智能体（Agent），专门针对酒馆战棋对局场景实现「画面实时捕获→战局视觉解析→策略推理决策」全链路自动化分析，解决玩家对局中信息过载、最优抉择难判断、数值记忆成本高的痛点。
二、核心技术架构：VLM+LLM 双模型协同
VLM 视觉大模型（视觉感知层）
通过窗口截图捕获模块读取游戏对局画面，完成全画面多目标识别：场上随从、手牌、酒馆可购买随从、英雄技能、金币数量、对手阵容、buff / 词条特效、圣盾 / 剧毒 / 嘲讽等关键词视觉提取；无需手动录入任何游戏信息，纯视觉自动解析当前完整战局状态。
LLM 语言大模型（策略推理层）
将 VLM 识别出的标准化战局信息送入语言大模型，依托酒馆战棋完整版本知识库、随从数值体系、流派运营逻辑、经济节奏、对战博弈逻辑做多层推理：计算随从战力、推演升本节奏、搭配阵容联动、预判对手威胁、给出最优购买 / 刷新 / 升本 / 卖随操作建议，输出可直接落地的对局决策。
Agent 调度中枢
串联截图捕获、视觉识别、大模型推理、结果输出交互窗口，提供一键调用、快速扫描、参数配置一体化调度能力。

三、界面功能模块详解
1. 核心操作按钮区
Capture & Decide（捕获并决策）
工具核心功能键：一键执行完整 Agent 流程 —— 自动截取游戏窗口画面 → VLM 解析全部战局视觉信息 → 送入 LLM 完成策略推演 → 在下方文本框输出完整对局分析与最优操作方案，是完整战局深度分析入口。
Fast Scan（快速扫描）
轻量化识别模式，跳过复杂长文本推理，仅用 VLM 快速提取场上关键随从、金币、核心 buff 等基础信息，适合快速浏览当前场上关键元素，响应速度更快。
Settings（设置）
全局配置面板：可调整截图捕获范围、模型调用参数、窗口置顶、自动隐藏、大模型接口密钥、酒馆版本素材库切换、输出文本格式等自定义参数。
Refresh? / Level? / Buy? 专项快捷按钮
细分场景定向分析入口：分别针对刷新酒馆、升本节奏、随从购买三类高频操作单独调用模型推理，精准回答单一维度博弈问题，相比完整扫描更轻量化。
Ask 自定义提问输入框
自由输入自然语言问题，结合当前捕获的画面战局信息，让 LLM 针对性解答自定义疑问（如 “当前经济要不要升本”“这个随从值不值得拿”），实现交互式对局咨询。
2. 运行状态区（Ready）
实时展示 Agent 当前就绪状态，提示截图模块、VLM/LLM 模型接口是否连通，直观判断工具是否可正常执行捕获与推理。
3. 日志 & 输出文本框
所有视觉识别日志、模型推理过程、完整决策建议、操作提示均在此输出；同时展示工具使用指引，内置窗口置顶、捕获自动隐藏等辅助功能说明。
4. 辅助实用特性：窗口置顶捕获
支持 Pin top-right 窗口置顶控制，工具界面可悬浮在游戏窗口上层不遮挡对局；配套 Hide while capturing 捕获自动隐藏开关，截图时自动隐藏本工具界面，避免 UI 遮挡游戏画面干扰 VLM 视觉识别，保证画面识别准确率。
四、核心能力与使用流程
前置就绪：打开炉石酒馆战棋对局，点击run_agent.bat，确认顶部显示「Ready」就绪；(必须下载卡牌数据仓库链接：https://github.com/DLW20051207/-agent）
一键全流程分析：点击 Capture & Decide，Agent 自动完成截图→视觉识别→大模型策略推理；
读取决策方案：底部文本框输出完整分析，包含阵容搭配、经济运营、购买优先级、对战风险提示；
轻量化补充查询：使用 Fast Scan 快速识别场上卡牌，或通过 Ask 输入自定义问题针对性咨询；
自定义参数调优：进入 Settings 配置模型接口、截图范围、窗口显示规则，适配不同分辨率游戏画面。
五、工具优势
纯视觉输入，零手动录入
依托 VLM 视觉大模型，仅靠截图就能读懂游戏画面，无需手动输入随从、金币、buff 等信息，降低使用门槛；
双模型分层推理，决策更专业
VLM 负责客观画面信息提取，LLM 结合海量酒馆战棋对局知识做策略博弈，兼顾画面识别精度与策略深度；
轻量化 / 完整两种扫描模式兼顾效率
完整深度推理、快速轻量化扫描双模式，满足精细复盘与快速浏览两种使用场景；
悬浮置顶不干扰游戏
配套窗口隐藏、置顶功能，工具界面可和游戏画面共存，截图时自动规避遮挡，不影响正常游玩；
交互式自然语言问答
支持自定义提问，结合实时战局画面给出个性化博弈建议，灵活应对各类复杂对局场景。
六、适用场景
新手玩家学习运营节奏、阵容搭配，快速理解版本强势流派；
进阶玩家对局实时辅助，快速计算最优操作、规避决策失误
